To highlight a text in bold in markdow use the following syntax

`**Sample Text**`

## italicizing-text-in-markdown
To italicize a text in markdow use the following syntax

`*Sample Text*`

## creating-a-link-in-markdown
To creat a link in markdown, use the following syntax

```
[Link name](linkPath)
[Link to google.com](https://www.google.com)
```

eg: [Link to google.com](https://www.google.com)
